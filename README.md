# Koffiewallet-demo

Een eenvoudige, werkende demonstratie van een digitale Koffiewallet voor **De Koffie Halte**.

## Wat deze eerste versie kan

- Een zichtbaar walletsaldo tonen
- Demo-tegoed toevoegen
- Een koffie-aankoop van het saldo aftrekken
- De laatste transacties weergeven
- De gegevens lokaal in de browser bewaren
- De demo terugzetten naar het beginsaldo

## Belangrijk

Dit is uitsluitend een demonstratie. Er is nog geen echte betaalprovider, klantenaccount, database, QR-code of bankkoppeling aanwezig. Er worden geen echte betalingen verwerkt.

## Bestanden

- `index.html` — de inhoud en structuur van de webapp
- `style.css` — de volledige vormgeving
- `app.js` — saldo, transacties en lokale opslag

## Lokaal bekijken

Open `index.html` in een browser. Voor normaal gebruik is geen installatie nodig.

## Mogelijke vervolgstappen

1. De demo online publiceren
2. Huisstijl en logo verder uitwerken
3. Werken met unieke klantaccounts
4. QR-code voor snel betalen toevoegen
5. Een veilige database en beheerdersscherm bouwen
6. Later een echte betaalprovider koppelen
